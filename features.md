## 👤 User Panels & Features

The platform provides three distinct, role-based dashboards with a modern, responsive SaaS-style interface.

### 1. Admin Panel

The **Admin Panel** is the centralized control hub for institutional staff, offering complete lifecycle management, financial oversight, and system administration.

#### Student Enrollment & Registration
- [x] Individual and batch (CSV) student enrollment
- [x] Auto-generation of Student ID and default login credentials
- [x] Full student record management (edit, suspend, withdraw, alumni status)
- [x] Complete audit history for all changes

#### Fee Management & Accounting
- [x] Configure fee structures by program, semester, and category
- [x] Manual adjustments (discounts, waivers, scholarships) with notes
- [x] Financial dashboard with real-time collection trends
- [x] Exportable financial reports (Excel + PDF)
- [x] Automatic blocking of registration for students with dues

#### Timetable Management
- Create and publish timetables with automatic conflict detection
- Mid-semester updates with instant notifications
- Export as PDF and iCalendar (.ics)

#### Notice Board & Announcements
- [x] Publish targeted announcements (by role or department)
- [x] Support for pinned/critical notices
- [x] Multi-channel notifications (Email + SMS + In-App)
- [x] Searchable announcement archive

#### Analytics & Reporting
- Institutional metrics: enrollment, fee collection, attendance rate, platform usage
- Interactive dashboards with drill-down
- Scheduled automated reports
- Full audit logs with advanced filtering

#### System Controls
- User account management (create/suspend/delete)
- Storage usage and quota management
- Platform configuration and security settings

---

### 2. Faculty Panel

The **Faculty Panel** equips teachers with powerful tools for course delivery and student evaluation.

#### Course Management
- [x] Create and manage courses (title, code, credits, syllabus, resources)
- [x] View enrolled student roster with contact details
- [x] Archive/deactivate courses at semester end

#### Assignment Creation & Grading
- [x] Create assignments with deadlines, marks, and rubrics
- [x] Unified submission inbox with late status detection
- [x] Inline grading with marks and written feedback
- [x] Bulk grade export (CSV/Excel)
- [x] Automatic student notification on grade publication

#### Attendance Tracking
- [x] Mark attendance (Present, Absent, Late, Excused)
- [x] Bulk CSV import support
- [x] Per-course and per-student attendance summaries
- [x] Automatic low-attendance alerts

#### Student Feedback & Evaluation
- Create structured feedback forms
- View aggregated analytics and charts
- Individual student evaluation tracking

#### Faculty Dashboard Analytics
- Class average marks and grade distribution
- Attendance trends and submission rates
- Late submission analytics

---

### 3. Student Panel

The **Student Panel** is a clean, self-service dashboard giving students full control over their academic journey.

#### Dashboard Overview
- [x] Personalized view with GPA trend, attendance risk, deadlines, and fee status
- [x] Quick-access cards for all major sections

#### Course Registration
- [x] Browse and filter available courses
- [x] Real-time seat availability
- [x] Add/drop/swap with conflict detection
- [x] Auto-generated weekly timetable (PDF download)

#### Assignment Submission
- [x] View assignments with deadlines and instructions
- [x] Submit files (PDF, DOCX, ZIP, JPG/PNG – max 25 MB)
- [x] Automatic late flagging
- [x] View marks and teacher feedback after grading

#### Attendance Tracking
- Real-time attendance percentage per course
- Color-coded risk indicators
- Detailed date-wise history
- Automatic alerts for low attendance

#### Fee Payment
- [x] Detailed fee breakdown by semester
- [x] Online payment via JazzCash, EasyPaisa, and cards
- [x] Digital PDF receipt generation and email
- [x] Payment history with filters
- [x] Registration blocked until dues are cleared

#### Student Analytics
- [x] GPA trend charts
- [x] Attendance risk indicators
- [x] Assignment completion tracking
- [x] Fee status summary

#### Notifications
- [x] In-app notification bell
- [x] Multi-channel alerts (Email + In-App) for deadlines, grades, fees, and announcements
